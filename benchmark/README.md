# TE NIMS Benchmarks

This directory packages the public benchmark artifacts that back TE NIMS
evaluation claims.

Current benchmark bundle:

- `oda-bench-v0/` — 52-case internal Operational Decision Accuracy benchmark

The benchmark is published as a **developing internal benchmark**, not a
peer-reviewed standard. Its purpose is transparency: judges and reviewers
should be able to inspect the case set, understand the scoring posture, and
see the distinction between direct-model and full-harness measurements.
